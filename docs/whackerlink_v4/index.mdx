# WhackerLinkv4: Getting Started

WhackerLinkv4 is a highly versatile framework for emulating P25 digital radio systems.
The most common use of this framework is within FiveM for radio communications through the FiveM script and the Dispatch Console application.
WhackerLinkv4 (also known as WhackerLinkServer) can be hosted in a couple of different ways.
