# Welcome to WhackerLink

**Seamless Communication for Ultimate Immersion.**

WhackerLink is a highly versatile framework for emulating P25 digital radio systems. The most common use of this framework is within FiveM for radio communications through the FiveM script and the Dispatch Console application.
