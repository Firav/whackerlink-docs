# WhackerLinkFiveM: Getting Started

:::danger
This content is AI generated and may not be accurate. This page is still under development.
:::

WhackerLinkFiveM is a FiveM resource that provides realistic P25 digital radio communications for roleplay servers. This client-side interface connects to WhackerLink v4 servers to enable authentic radio experiences within FiveM.

## ðŸ“‹ Prerequisites

Before installing WhackerLinkFiveM, ensure you have:

- A running WhackerLink v4 server instance
- FiveM server with appropriate permissions
- Basic understanding of FiveM resource management

:::warning
You must have a WhackerLink v4 server to use WhackerLinkFiveM. If you do not have a server, consider using a managed hosting service like [Smoky Ridge Solutions](https://smokyridge.solutions/) or follow the [self-host installation guide](/docs/whackerlink_v4/installation) to set up your own server.
:::
